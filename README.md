# Scratchpad

A webextension that uses a browserAction (toolbar button) to display a popup
with a textarea.

## Install

### Firefox

Install the `web-ext` helper:

    npm install -g web-ext

Clone the repository, then:

    web-ext run


### Chrome

Navigate to [chrome://extensions/](chrome://extensions/) and enable `Developer
mode`.

Then click on `Load unpacked extension...`.
